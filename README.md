# EventX API (starter)

## Setup
1. Copy `.env.example` to `.env` and fill values (use MongoDB Atlas or local Mongo).
2. Install dependencies:
   ```bash
   cd api
   npm install
   ```
3. Run seed (optional):
   ```bash
   npm run seed
   ```
4. Start server:
   ```bash
   npm run dev
   ```
API base: http://localhost:8080/api
